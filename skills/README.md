# Skills

Skills are reusable workflow guides that help AI agents perform specific tasks effectively in this repository.

## Available Skills

| Skill | Description |
|-------|-------------|
| [maven-indexer](./maven-indexer/SKILL.md) | Query the local Maven/Gradle artifact index from the terminal. Use when you see an import but cannot find the class definition in the workspace, need to read the actual implementation of an internal library, find which artifact contains a class, discover SPI implementations, or inspect proto/resource files inside JARs. |
